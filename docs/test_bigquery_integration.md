# Documentation du script test_bigquery_integration.sh

Ce script permet de tester l'intégration entre le projet et BigQuery.

## Utilisation

```bash
bash scripts/test_bigquery_integration.sh
```

## Rôle
- Vérifie que les tables et datasets sont accessibles
- Teste l'insertion et la lecture de données dans BigQuery
- Affiche les résultats des tests

## Prérequis
- Connexion à BigQuery fonctionnelle
- Datasets déjà créés

## Résultat attendu
- Affichage des tests passés ou échoués
- Diagnostic rapide de l'intégration BigQuery
